### ⚠️ Configuración de Credenciales JMeter

El plan de pruebas incluye un paso de autenticación con credenciales que podrían no ser válidas en tu entorno local. Si el test falla en el login, por favor edita el archivo `Pruebas JMeter TDH.jmx` manualmente:

* **Línea 154:** Actualizar el valor del `username`.
* **Línea 157:** Actualizar el valor del `password`.
